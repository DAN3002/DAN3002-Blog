Link a package to use it locally

```bash
cd ~/path/to/package
npm link
```

Unlink a package

```bash
cd ~/path/to/package
npm unlink
```

Remove from global npm list

```bash
npm rm --global <package-name>
```

Check if the package is removed

```bash
npm list --global
```
